Installation
=======================

1. Open the file CONSOLAS.TTF
2. Install the font.
3. Copy the theme file to the Xcode's ``FontAndColorThemes`` folder ``(~/Library/Developer/Xcode/UserData/FontAndColorThemes/)``.
4. Restart Xcode.
5. Apply theme.
